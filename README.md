## About The Project

<!-- GETTING STARTED -->

## Getting Started

### Requirements

An installed rust-toolchain. Look at <https://rustup.rs/>.

### Installation

1. Clone the repo

   ```sh
   git clone https://github.com/pingoin-tech/home_center.git
   ```

2. Compile from source

    ```sh
    cargo build --release
    ```
